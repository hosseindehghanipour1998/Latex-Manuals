\section{Requirements and How To Run The Code}

\subsection{Requirements}
Before running this code please make sure you have these tools:\\
\begin{itemize}
\item Anaconda 4.10.3 or higher.
\item Python 3.7.9 or higher.
\item OpenCV 4.0.1 or higher.
\item Numpy 1.19.2 or higher.
\item Scipy The newer version, the better.
\item Matplotlib The newer version, the better.
\item Spyder 4.1.5 or higher.
\end{itemize}

In order to do the installation easier and avoiding any conflicts with the packages you already have, it is suggested to use \href{https://www.anaconda.com/}{Anaconda} as your package management.

\subsection{Running The Code}
In order to run the code, all you have to do is to open the \hl{Main.py} in \textbf{Spyder} and then run the code by pressing F5 (or whatever your IDE settings is set to). If you want to see the source code of the functions, you can read \textbf{Main.py} file. You can take a look at the code to see how the mathematics is done. All parts of the code are well commented.


\subsection{Directories}
This project is consisted of Four directories names:\\
\begin{itemize}
\item \textbf{\_\_pycache\_\_} which is Python cache while we run the code.
\item \textbf{Inputs} which contains the images we are using as input (8 frames).
\item \textbf{Outputs} which contains the output images that are produced by our code.
\end{itemize}
